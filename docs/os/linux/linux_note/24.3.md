## **24.3 备份要点**

### **24.3.1 备份资料考虑**

* 推荐备份目录
    * **/etc**
    * **/home**
    * **/root**
    * **/var/spool/mail/**
    * **/var/spool/cron/**
    * **/var/spool/at/**
    * **/var/lib/**

* 不需要备份目录
    * **/dev**
    * **/proc**
    * **/sys**
    * **/run**
    * **/mnt**
    * **/media**
    * **/tmp**
